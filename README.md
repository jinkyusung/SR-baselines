# Sequential Recommendation Baselines

Sequential recommendation baselines for VIPLab research projects.  

- JODIE (KDD 19)  
- LT-OCF (CIKM 21)  
- GRU-ODE or GNG-ODE (CIKM 22)  
- DGEL (SIGIR 23)
- E-NSDE (NeurIPS 24)
- FMRec (IJCAI 25)
- TGODE (KDD 25)  

## Dataset Information

All datasets we use are available in the [RecBole Library](https://github.com/RUCAIBox/RecSysDatasets).  
You can download the `*.inter` datasets directly from the [RecBole Drive](https://drive.google.com/drive/folders/1so0lckI6N6_niVEYaBu-LIcpOdZf99kj).
